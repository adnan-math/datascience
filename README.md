# datascience

A collection of foundational notebooks and notes covering core concepts in Python for scientific computing, numerical methods, and data visualization.

This repository reflects early-stage work focused on building strong fundamentals in numerical computing and data handling, which form the basis for advanced machine learning and data science workflows.

---

## 📌 Overview

The repository includes exploratory notebooks and notes on:

- Python data structures and indexing
- Numerical computing with NumPy
- Basic data visualization techniques
- Mathematical concepts relevant to machine learning (e.g., activation functions)

These materials were developed as part of hands-on learning and practice in computational mathematics and scientific programming.

---

## 📂 Repository Structure

- `numpy_practice-checkpoint.ipynb`  
  Introduction to numerical computing using NumPy (arrays, operations, vectorization)

- `indexing and data structure-checkpoint.ipynb`  
  Python data structures and indexing techniques

- `Visulization-checkpoint.ipynb`  
  Basic data visualization using Python libraries

- `chilla_data-checkpoint.ipynb`  
  Dataset exploration and plotting exercises

- `session_Day_2-checkpoint.ipynb`  
  Practice notebook covering multiple foundational topics

- `Activation_fun.md`  
  Notes on activation functions used in machine learning
